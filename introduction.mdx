## What is Sushi Restaurant App?

Sushi Restaurant App is a modern mobile application built with Ionic and Angular that showcases a seafood restaurant menu. The app features a beautiful marine-themed UI and provides an intuitive way to browse dishes, view detailed information about ingredients, and manage menu items.

## Technology Stack

The Sushi Restaurant App is built with modern web technologies:

- **Framework**: Ionic Angular 8.0 with Angular 20
- **Mobile Runtime**: Capacitor 8.1 for native mobile capabilities
- **Styling**: Tailwind CSS with custom marine theme
- **Language**: TypeScript for type-safe development
- **Testing**: Jasmine and Karma for unit testing
